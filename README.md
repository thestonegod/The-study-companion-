# The-study-companion-
# Study Tracking & Reflection App

## Overview
A personal project of a study companion that tracks study time and allows you to reflect on what they learned, 
and monitor consistency over time.

## Main Layout

[ Header ]
- App Title: Study Tracker
- Date / Greeting

[ Study Input Section ]
- Input: Study Subject
- Input: Study Topic
- Input: Duration (minutes)
- Button: "Add Study Session"

[ Reflection Section ]
- Textarea: "What did I learn today?"
- Textarea: "What was difficult?"
- Button: "Save Reflection"

[ Progress Section ]
- Total study sessions
- Total minutes studied
- Simple progress indicator (text-based for MVP)

[ History Section ]
- List of previous study entries:
  - Date
  - Topic
  - Duration

[Footer]
- Motivational line / quote
